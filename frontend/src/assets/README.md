# Assets Directory

## Hall Background Image

To use the majestic hall background image, please:

1. Add your hall background image to this directory
2. Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
3. Recommended filename: `hall-background.jpg` (or update the filename in `hero-video.component.ts`)
4. Recommended size: 1920x1080 or larger for best quality
5. The image should be a majestic hall with futuristic/fantasy architecture in white, silver, and deep blue tones

### Current Image Reference

The component expects the image at: `assets/hall-background.jpg`

Update the `hallBackgroundSrc` property in `hero-video.component.ts` if you use a different filename.
